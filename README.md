# In-Silico-PCR-Agarose-Gel-Simulator
In Silico PCR Agarose Gel Simulator is an app that enables any user to simulate the expected bands of any PCR products of interest running in a designable agarose gel (you can select % agarose, voltage and run time). The goal is to have an accurate PNG file which can be used for comparing with a real agarose gel. Also the 4 most used ladders in the market for nucleic acid molecular sizing can be selected. It allows only 1 PCR product per well.
